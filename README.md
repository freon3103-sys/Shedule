# Shedule
Парсер маршрутов общественного транспорта Самары с сайта samaratrans.info. Умеет сравнивать старую и новую версии расписания, выбирать нужные маршруты галочками и просматривать расписание в интерфейсе.
Для работы создайте в папке с программой файл Links.txt и впишите в него ссылки на маршруты с подробным расписанием (каждая ссылка с новой строки).

Пример:
https://www.samaratrans.info/wiki/index.php?title=Самара_автобус_67
